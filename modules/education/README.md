# DevShield-AI Education & Policy Engine

> _"Not just a warning—learn why, and how to fix it. DevShield-AI teaches as it protects."_

This module is part of the [DevShield-AI](../../README.md) suite, educating developers about secret risks and enforcing security policies with friendly, actionable messages.

## Folder Structure
```
education/
├── tips.py           # Tips & explanations logic
├── policy_rules.py   # Simple policy enforcement engine
└── README.md         # Documentation and instructions
```


---

## What It Does

- **Explains risks**: why a secret is dangerous, how to fix it, and best practices
- **Policy engine**: block, warn, or enforce environment variables
- **CLI and popup support**: educates in the terminal, IDE, or browser
- **Integration-ready**: works with Developer Guard, AI Risk Engine, and all DevShield-AI clients
